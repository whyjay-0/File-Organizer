# File Organizer

A simple application built using Python and Tkinter to organize files in a folder by their types (images, documents, videos, and audio).

Select a folder, and the program will move files into the appropriate subfolders.

## Features
- Select a folder to organize.
- Automatically organizes files into predefined categories (Images, Documents, Videos, Audio).

## Supported Operating Systems
- **Windows** (Currently tested on Windows 11)

### Requirements:
- **Python** 3.6 or later

## Installation
### Method 1: Clone the Repository and Run the Python Script
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/file-organizer.git
2. **Navigate to the project directory:**
   ```bash
   cd file-organizer
3. Run the script
  ```bash
  python organize.py
  ```

### Method 2: Download and Run the Executable
1. **Go to the [Releases page](https://github.com/whyjay-0/File-Organizer/releases)**
2. Download the precompiled executable for your operating system
3. **Run the executable:**
   - Double-click the `File Organizer.exe` file to run the application.

## Usage
1. Select a folder you want to organize using "Browse" button.
2. The program will create folders and sort each file based on their file type.
