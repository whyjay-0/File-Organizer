# File Organizer

A simple application built using Python and Tkinter to organize files in a folder by their types (images, documents, videos, and audio).

Select a folder, and the program will move files into the appropriate subfolders.

## Features
- Select a folder to organize.
- Automatically organizes files into predefined categories (Images, Documents, Videos, Audio).

## Installation
1. Clone the repository: `git clone https://github.com/your-username/file-organizer.git`

## Usage
Run the Python script:
```bash
python organize.py
